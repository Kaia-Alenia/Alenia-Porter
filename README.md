# Alenia Porter v4.2 🐧✨

**The ultimate media optimizer for indie game developers.**

Alenia Porter is a professional tool designed to streamline your workflow when developing for **Ren'Py** and **Godot**. It automatically converts and optimizes images, videos, and audio files to ensure maximum performance and minimum disk space.

## ✨ Features
- **Engine Optimized:** Tailored conversion settings for Ren'Py (WebM/WebP) and Godot (OGV/WebP).
- **Dynamic Themes:** Fully customizable UI. Swap between Pink, Dark, Light, and Eye Care themes with a single click.
- **Kaia Assistant:** Your personal companion that guides you through the conversion process with dynamic states.
- **Ultra-Slim Popups:** Efficient and non-intrusive information windows.

## 🎨 Customization
Alenia Porter is designed to be yours. You can:
- **Create Themes:** Add your own colors and sprites via simple JSON files.
- **Support Us:** Integrated support window to help Alenia Studios continue creating free tools for the community.

## 🚀 Getting Started
1. Run `python3 main.py`.
2. Select your target engine.
3. Choose your preferred output format.
4. Select the folder containing your raw assets and let Kaia do the rest!
