# Alenia Porter v3.0

### Media Optimizer for Game Engines (Ren'Py / Godot)

Alenia Porter is a tool designed to automate the conversion and optimization of multimedia assets, ensuring that final files are compatible and lightweight for independent projects.

## Contributions and Development

We invite the community to improve Alenia Porter. You can collaborate on:
* Interface (UI): Improve design and user experience.
* Logic: Optimize conversion processes and file handling.
* Languages: If you want to see Alenia Porter in your language, you can edit the locales.json file and submit your changes.

### Roadmap
We are currently working on extending support to Unity and other engines. Due to our current workstation specifications (8GB of RAM), testing on heavier engines is done progressively.
